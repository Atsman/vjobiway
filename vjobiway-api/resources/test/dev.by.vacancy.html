ITOS приглашает в свою команду Java developer.

Требования:

  • Уверенное знание технологий: Java, Spring, Hibernate, OSGi, REST, ActiveMQ.
  • Уровень владения английским языком от “Intermediate”.
  • Умение работать в команде, стремление к качеству, нацеленность на результат.

Мы предлагаем:

  • Интересную работу в дружной команде профессионалов.
  • Широкие возможности для карьерного и профессионального роста.
  • Возможность изучения английского языка за счет компании.
  • Комфортные условия работы.
  • Адекватный менеджмент и увлекательные проекты (мы прислушиваемся к идеям сотрудников и даем возможности для их реализации).
  • Интересные корпоративные мероприятия.
  • Конкурентную заработную плату (размер обсуждается индивидуально по итогам собеседования).

Требования к кандидату

  • Опыт: 3 года
  • Технологии: java, Spring, hibernate
